🚀 Mis Proyectos como QA Engineer

En mi trayectoria como QA Engineer he trabajado con diversas herramientas y enfoques para garantizar la calidad del software. Aquí algunos de los entornos y tecnologías con los que he trabajado:

Postman

Automatización de pruebas de API REST, validaciones de respuestas, assertions y documentación.

Selenium

Automatización de pruebas funcionales en navegadores, flujos de usuario y pruebas de regresión.

PyCharm

Automatización de pruebas en Python con Pytest, ejecución de scripts y análisis de cobertura.

Android Studio

Pruebas de interfaz de usuario en apps móviles, revisión de usabilidad y experiencia de usuario.

Excel para QA

Análisis de datos de pruebas, tablas dinámicas, seguimiento de bugs y priorización.

Git y GitHub

Control de versiones, documentación de proyectos, manejo de ramas y seguimiento de cambios.
